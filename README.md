Qahwa
====

for WHAM. this is in development.

## Install

clone this repository on your machine.

    $ git clone git@moca:niina/Qahwa.git

At first, you should initialize submodules.

    $ git submodule update --init --recursive

use CMake.

    $ cd build
    $ cmake ..
    $ make
